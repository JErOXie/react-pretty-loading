# react-loading
用于在react中显示加载动画

### 使用指南

在导入组件时只需要导入需要的组件，这样在打包时可以缩小代码体积
```
npm install --save react-pretty-loading
import {
  Ball8bits
} from "react-pretty-loading";
<Ball8bits loading center />
```

### 动画来源
